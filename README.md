**Задание 7.**
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Для класса LineSegment перегрузить операции:

! — вычислить длину диапазона.

++ — расширить отрезок на 1 вправо и влево.

Приведение к int (неявное) — целая часть координаты x.

Приведение к double (явное) — координата y.

- (левостороннее) — уменьшить координату x.

- (правостороннее) — уменьшить координату y.

< — проверка пересечения отрезков.

**Обьяснение:** !line → длина отрезка.

line++ → расширение отрезка.

(int)line → целая часть X.

(double)line → значение Y.

line - 10 → сдвиг X влево.

10 - line → сдвиг Y вниз.

line1 < line2 → проверка пересечения.
